# Modelo de Prompt para Gerar Código TensorFlow.js

Para obter de uma IA um script JavaScript completo que treine e salve um modelo com TensorFlow.js, descreva seu pedido seguindo esta estrutura:

---

## 1. Contexto e Objetivo
- Explique o propósito do modelo (classificação de imagens, previsão de séries temporais, etc.).  
- Indique o formato dos dados (CSV, imagens em pastas, JSON, API).  
- Defina o ambiente alvo (navegador ou Node.js).

---

## 2. Especificação do Dataset
- Localização (URL, caminho local ou armazenamento em nuvem).  
- Formato e shape dos dados (por exemplo, imagens 28×28, CSV com colunas “feature1,feature2,label”).  
- Estratégia de pré-processamento (normalização, one-hot encoding, data augmentation).

---

## 3. Arquitetura do Modelo
- Tipo de rede (MLP, CNN, RNN, Transformer).  
- Número de camadas, unidades/neuronios por camada e funções de ativação.  
- Camada de saída e função de perda (por exemplo, softmax + categoricalCrossentropy).

---

## 4. Configurações de Treinamento
- Otimizador e taxa de aprendizado (por exemplo, adam, 0.001).  
- Número de épocas e tamanho de batch.  
- Métricas de avaliação (acurácia, perda, F1-score).

---

## 5. Salvamento e Exportação
- Indique como salvar o modelo (model.save('localstorage://meu-modelo') ou em arquivo para Node.js).  
- Explique o formato desejado (.json + binários .bin).

---

## 6. Exemplo de Prompt Completo

\`\`\`markdown
Crie um arquivo JavaScript usando TensorFlow.js em Node.js que:

1. Carregue o arquivo CSV “dataset.csv” com colunas “x1,x2,x3,label”.
2. Normalize as features para intervalo [0,1].
3. Construa uma rede MLP com:
   - Entrada de 3 unidades
   - Duas camadas ocultas de 64 neurônios (ativação relu)
   - Camada de saída de 3 neurônios (ativação softmax)
4. Compile o modelo com otimizador adam (learningRate=0.001) e loss categoricalCrossentropy.
5. Treine por 50 épocas, batchSize=32, exibindo loss e accuracy a cada 5 épocas.
6. Salve o modelo no disco local em “./model_tfjs”.
7. Inclua comentários explicando cada etapa do código.
\`\`\`

---

## 7. Perguntas de Refinamento
- Qual é o tamanho do seu conjunto de dados?  
- Você precisa de data augmentation (no caso de imagens)?  
- Vai usar validação cruzada ou separar apenas treino/teste?

---

## 8. Próximos Passos Que Você Pode Não Ter Perguntado
- Otimização de hiperparâmetros (Hyperparameter Tuning) usando tensorflow-models/tfjs-hyperparameter.
- Conversão de um modelo Python (Keras) para TensorFlow.js com tensorflowjs_converter.
- Deploy do modelo no navegador com WebGL para acelerar inferência.
- Quantização do modelo para reduzir tamanho e aumentar performance em dispositivos móveis.

Você pode ajustar esse template conforme seu caso de uso e, com isso, qualquer IA generativa saberá exatamente como criar seu modelo em TensorFlow.js.

# Fontes de CSV Prontos para Modelos de IA

Se você precisa de arquivos CSV já estruturados para treinar seus modelos, existem diversos repositórios e plataformas que oferecem dados abertos, gratuitos e de qualidade. Abaixo estão as principais opções:

---

## 1. Plataformas de Datasets Abertos

- Kaggle: Milhares de conjuntos de dados em diversos domínios. Filtre por fileType=csv.
- UCI Machine Learning Repository: Datasets acadêmicos bem documentados.
- Google Dataset Search: Ferramenta de busca global de datasets em CSV.
- Data.gov / Dados.gov.br: Portais governamentais com centenas de planilhas CSV.
- Shaip Open Datasets: Dados prontos para ML/NLP/CV em CSV.

---

## 2. Comunidades e Repositórios Colaborativos

- data.world: Plataforma colaborativa com vários formatos.
- OpenML: Mais de 20.000 datasets para Machine Learning.
- GitHub: Procure repositórios “awesome-datasets” para listas curadas.

---

## 3. Como Escolher o Dataset Certo

1. Defina o objetivo (classificação, regressão, séries temporais).
2. Verifique tamanho e qualidade (número de amostras, valores faltantes).
3. Analise licenças de uso.
4. Confira documentação (schema, descrição de variáveis).

---
